Инструменты для реализации:
Python 3.8+ — язык программирования
pyTelegramBotAPI — библиотека для создания Telegram-ботов
Telegram Bot API — протокол взаимодействия с Telegram
datetime — работа с датой и временем (встроенный модуль)
threading — для фоновых напоминаний (встроенный модуль)
time — задержки и таймеры (встроенный модуль)

Проект состоит из:
1. Бот (bot.py) — основной файл с кодом

Обработчик команды /start

Обработчики кнопок (Расписание, Оценить урок, О школе, Контакты)

Система оценки уроков (звёзды 1-5)

Хранилище оценок (ratings)

Фоновая система напоминаний

2. Клавиатура — ReplyKeyboardMarkup с 4 кнопками

3. Inline-клавиатура — кнопки с оценками 1-5 ⭐

4. Система напоминаний — проверка каждые 30 секунд, отправка в субботу в 18:40

5. README.md — документация с описанием функций
